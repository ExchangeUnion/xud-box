# xud-box
Launch a fresh/new xud-simnet instance with 1 command.

## Install
Clone the repository.
`git clone https://github.com/ExchangeUnion/xud-box.git ~/xud-box`

## Usage
```
cd ~/xud-box
vagrant up
```

## Dependencies
* [virtualbox](https://www.virtualbox.org)
* [vagrant](https://www.vagrantup.com)
